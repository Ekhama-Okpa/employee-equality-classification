# employee-equality-classification
**Classifying "equality class" of each employee based on their "equality score"**

An excel file has been generated based on data collected on employee compensation.

The table contains three columns including an "Equality Score" column containing integers ranging between -100 and +100.

A fourth column, "Equality Class", classifies the equality score into 3 categories:
Fair (+-10)
Unfair (<-10 AND >10)
Highly Discriminative (<-20 AND >20)

The analysis was conducted using different methods. The method uploaded was via the openpyxl module using Python.
